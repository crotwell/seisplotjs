SeismogramLoader

src/seismogramloader.ts

Loads seismograms based on queries to Station and Event web services. Uses the traveltime web service to create times for data query. Default is P-30 sec to S+60 sec.

new SeismogramLoader(stationQuery: (StationQuery | Array<Network>), eventQuery: (EventQuery | Array<Quake>), dataselectQuery: DataSelectQuery?)
Parameters
stationQuery ((StationQuery | Array<Network>)) query to find channels
eventQuery ((EventQuery | Array<Quake>)) query to find events
dataselectQuery (DataSelectQuery?) optional additional parameters for seismogram query

Seisplotjs 3.0.0-alpha.4

Seisplotjs is a collection of javascript routines for requesting, manipulating and plotting seismic data.

Seisplotjs is available via npm, by npm i --save seisplotjs.

Full source at Github.

A standalone js file with all dependencies included is available at http://www.seis.sc.edu/downloads/seisplotjs/. This is used by the tutorial and examples.

Documentation for previous versions is archived at http://www.seis.sc.edu/software/seisplotjs
